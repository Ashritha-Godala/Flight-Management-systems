#  Flight Management System

A desktop application built with **Java** and **Swing** that simulates a flight management system.  
This project demonstrates core concepts of **object-oriented programming**, **GUI development**, and optional **database integration**.

---

## Features
- Add, update, and delete flight records  
- Manage passenger details and bookings  
- Search and filter flights by destination, date, or flight number  
- User-friendly Swing-based interface  
- Data validation and error handling  

---

##  Tech Stack
- **Language:** Java  
- **GUI Framework:** Swing  
- **Database (optional):** MySQL / SQLite (update based on your implementation)
